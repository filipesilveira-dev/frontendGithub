# Tecnologias e técnicas utilizadas

- **React**: framework javascript baseada em componentes reutilizáveis
- **Typescript**: linguagem de programação tipada baseada em javascript
- **CSS Modules**: arquivos css nos quais o className são definidos localmente
- **Axio**: cliente HTTP baseado em promessas para o Node.js e para o navegador. Faz as requisições HTTP (substitui o  método fetch)
- **API (crudcrud)**: fornece uma API rest que utiliza o protocolo HTTP para comunicações CRUD
- **Lucide**: biblioteca de ícones e símbolos para React
- **Mobile First**: estilização iniciada por telas menores
- **Lifting State Up**: estado *books* movido para App()
