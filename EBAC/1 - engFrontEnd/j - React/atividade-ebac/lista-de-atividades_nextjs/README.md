# Lista de tarfeas

Um aplicativo de gerenciamento de tarefas moderno, desenvolvido com **Next.js 14**, **TypeScript** e **Tailwind CSS**. O projeto foca em uma interface minimalista em escala de cinza, responsividade total e testes de unidade/integração robustos.

## 🚀 Funcionalidades

*   **Custom Hook:** Gerenciamento de estado lógico via `useContadorTarefas`.
*   **Responsividade:** Layout adaptável para Mobile, Tablet e Desktop.
*   **Interface Minimalista:** Design focado em UX com paleta de cores em tons de cinza.
*   **Testes Automatizados:** Cobertura de renderização e lógica de estado com Vitest/Jest e React Testing Library.

## 🛠️ Tecnologias Utilizadas

*   [Next.js](https://nextjs.org/) (App Router)
*   [TypeScript](https://www.typescriptlang.org/)
*   [Tailwind CSS](https://tailwindcss.com/)
*   [React Testing Library](https://testing-library.com/docs/react-testing-library/intro/)

## 📥 Instalação

1. Clone o repositório:
   git clone https://github.com/filipesilveira-dev/frontendGithub.git

2. Acesse a pasta do projeto:
   cd frontendGithub/tree/main/EBAC/1%20-%20engFrontEnd/j%20-%20React/atividade-ebac/lista-de-atividades_nextjs

3. Instale as dependências:
   npm install

4. Rode o projeto:
   npm run dev

O app estará disponível em http://localhost:3000
